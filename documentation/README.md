# Documentation

This folder will hold technical documentation for the project as it
matures.

For now, the main `README.md` at the repository root contains the
overview, setup instructions, and architecture summary.

Future content planned for this folder:

- Architecture deep-dive across the MCP service, ingest pipeline, and
  supporting services.
- Per-profile guides for consumer, publisher, and governance use cases.
- Operational runbooks for self-hosting.
- Reference for the MCP tool schemas.

Contributions to documentation are welcome — see the project
`CONTRIBUTING.md`.
