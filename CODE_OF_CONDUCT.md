# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1.

The full text of the Code of Conduct is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — through issues, pull requests,
discussions, or any other form of contribution — you agree to abide
by its terms.

## Reporting

Instances of behavior that violate the Code of Conduct may be reported
by contacting the project team at **conduct@openarx.ai**.

All complaints will be reviewed and investigated promptly and fairly.
The project team is obligated to respect the privacy and security of
the reporter of any incident.

## Scope

This Code of Conduct applies within all project spaces — the source
repository, issue tracker, discussions, and any other communication
channels used by the project — and also applies when an individual is
officially representing the project in public spaces.
